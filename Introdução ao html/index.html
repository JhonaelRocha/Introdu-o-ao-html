Estrutura Básica

Início

A estrura básica do HTML é iniciada pela tag "html", dentro dela temos uma cabeça ("head") e um corpo ("body"),aonde a cabeça ("head") são coisas que vão ser carregadas anter de exibir para o usuário, e o corpo ("body"), o que é exibido para o usuário. Um exemplo de algo definido no "head" é o título da aba do seu site, através da tag "title".

Tags

Início

As tag HTML são usadas para definir diferentes tipos de elementos na página
Exemplos :

  • "h1" : Formata o texto para título com níveis, aonde quanto maior o nível menor o título fica.
    Exemplos : h1, h2, h3..
  • "i" : Formata o texto para itálico.
  • "strong" : Formata o texto para negrito.
  • "input type="text" " : Cria uma caixa para inserir textos;
  • "blockquote" : Formata o texto para uma citação.

Atributos de Tags

Início

No HTML, os atributos de tag fornecem informações adicionais sobre os elementos e ajudam a controlar seu comportamento e aparência.
Exemplos :

  • "input type="text" " : O atributo "type" da tag "input" está definido para "text". Cria uma caixa para inserir textos;
  • "input type="number" " : O atributo "type" da tag "input" está definido para "number".Cria um caixa para inserir números;
  • "input type="color" " : O atributo "type" da tag "input" está definido para "color".Cria um caixa para inseris cores;
  • "style="color : blue;" " : O atributo "style" é usado para colocar comandos CSS direto no elemento;
  • "img scr="url_ou_caminho.com.br" width="150" " : O atributo "scr" é usado para passar um caminho ou url para a tag" img", que exibirá a imagem passada. O atributo "width" é usado para definir a largura a imagem.
  • "id" : Define uma id para aquele local.

Listas

Início

As listas são usadas para organizar e exibir itens relacionados em formato de lista.

  1. Ordenadas

    As listas ordenadas são usadas quando a ordem dos itens é importante. "ol" é a tag uasda para criar uma lista ordenada.

  2. Não Ordenadas

    As listas ordenadas são usadas quando a ordem dos itens não é importante. "ul" é a tag uasda para criar uma lista ordenada.

Ambas as listas precisam de tags filhas ("li") para adicionar membros.

Links

Início

Os links são usados para criar interações entre páginas
Para criar um link usamos a tag "a".

  • "a href="url" + nome " : Cria um link que leva para outro site.
  • "a href="url" target="_blank + nome"" : Cria um link que leva para outro site em uma nova aba.
  • "a href="url" target="_selg + nome"" : Cria um link que leva para outro site na aba atual.
  • "a href="# + id" target="_selg + nome"" : Cria um link que leva para outro local no site atual.
